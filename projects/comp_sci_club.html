Stockton University’s computer science club needed a front end design for their website so I volunteered to build something. Then I used this project to learn common development ops practices.

Services Provided

Project Tech Stack

  • PHP 7
  • MYSQL Database
  • Javascript ES6
  • HTML 5 & CSS 3 With Bootstrap

Voting Feature

The voting feature is implemented using PHP and MYSQL. A users have the ability to create club voting topics and then vote on the created topic. It is important that this feature keeps the voters anonymous and not allow them to cast duplicate votes. To solve this problem I've implemented an IP address based authentication system.
