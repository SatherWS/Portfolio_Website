For my software engineering final project at Stockton University, my team and I created a web application for Assurance Wireless. Assurance Wireless is a government assistance distributor of free mobile phones and cellular data.

Services Provided

  • Customer application processing
  • Realtime customer support chat (beta)
  • Geographic customer validation

Tech Stack

  • Python 3.7.5
  • MYSQL Database
  • Flask Framework
  • Javascript with AJAX
  • HTML 5 & CSS 3 with Bootstrap

On Site Application Processing

The main purpose of this project was to provide Assurance Wireless employees means of viewing and processing applications for government assistance. This application is configured to be easily viewable on tablet devices. This way remote recruiters are able to search for specific applications on site.

Customer Support Chatroom

We’ve implemented a live customer support chat that allows customers to ask questions to the customer support staff. Similar to the application management feature the chat feature is mobile responsive. This way remote recruiters are able answer customer support tickets if they have down time.
