Micro Server.

Web based productivity software hosted on local networks.


Project Source Code

Project Description

  • April. 1 2020 - June 1, 2020
    • •
  • Internal Web Application
    • •
  • colin sather

Micro Server is a collection of web applications that is meant to be hosted on an internal web server. The purpose of this software is to provide useful services to all client computers within a WiFi or LAN network.

Services Provided

  • Task Management
  • Shared Note Taking & Data Storage
  • Realtime Communication
  • Voting Services
  • Task & Moral Analytics

Project Tech Stack

  • PHP 7
  • MYSQL Database
  • Javascript ES6
  • CSS 3 (Mobile Responsive)
  • HTML 5

Distributed Note Taking Functionalities

One of the most important features of this application is the note station. With this tool a user can easily write notes to help them with a project. This feature allows the user to input html elements, which means you can add hyperlinks, images and video to any given note.

Note Creation Form

Below is a screenshot of the form used to submit notes to the Raspberry Pi server. All notes are public unless the user is logged in.

Viewing All Notes

We wanted to give users the option to use the note taking app without having to sign in. If a user does not log in they will still be able to write notes but the journals will be completly public.

Note Station Detail View

This is an example of what a Micro Server journal entry looks like.

Note Station Editor

Raw HTML data is stored in the database and is parsed in a human readable format thanks to PHP's nl2br function. This feature is simple and easy to use and is really beneficial when you want to book mark websites in a more organized manner.

Task Management Functionalities

This internal web application software also has a to do list feature. It's easy to use and inuitive you simply describe the task you need to complete, set a deadline for the task and recieve email reminders as the task deadline approaches.
