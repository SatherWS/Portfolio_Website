Project Description

    • •
  • Contracted Web Development Project
    • •

Skernedits.com is a website for a freelance videographer and copywriter. The purpose of this website is to establish a digital presence for the Skern Edits media company. I hosted this site using google firebase in order to support remote file uploads. File upload functionality is important to this company in order for its clients to upload documents to be evaluated for his services.
